# 🎹 Advanced Arpeggiator Plugin

A flexible MIDI arpeggiator that supports multiple playback modes, octave expansion, and expressive timing controls. 
All modes, sliders, and octave levels can be combined to create highly customizable melodic patterns.

# DelayFX
A customizable MIDI arpeggiator that transforms held notes into dynamic, evolving patterns. 
This plugin supports ascending/descending playback, multi‑octave spans, user‑defined note sequences, and variable note durations.
